Styles

namespace Styles

A set of predefined styles and style sheets that are used by specific components; these may be modified to apply a “theme”, which will generally update all styles in the DOM asynchronously; however, styles defined here are subject to change for now and it is not guaranteed that selectors and class names will continue to exist across versions.
