appendChildComponents

function appendChildComponents(type: ComponentFactory.CLevel, acceptFragments?: boolean): (target: typeof Component) => void;

Class decorator for a component class that expects child components (through appendChild method) of given type; the factory will then expand/wrap regular and observable objects, factories, classes, or components into the correct type; if acceptFragments is set, then components constructed from factories with the isFragmentFactory are passed in as-is; used on root component classes, should not need to be used in application code [decorator].
