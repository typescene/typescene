treeListRowParent

treeListRowParent?: TreeListRow;

Reference to the TreeListRow instance that created this row, if any, i.e. the parent row in the visual hierarchy; this property is set automatically by a parent constructor, and by the static methods arrayFromItems, and arrayFromBinding; otherwise it must be set before rendering.
