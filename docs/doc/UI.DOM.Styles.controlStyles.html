controlStyles: Stylesheet

Basic control theme stylesheet that uses colors from the .color object, enabled by default: applied as a live stylesheet at the page level for this instance; can be modified to adjust or add CSS reset styles, or disabled using the .disable() method if external CSS styles are already loaded.

This is the documentation page for “controlStyles (UI.DOM.Styles.controlStyles)”, part of the Typescene toolkit.
